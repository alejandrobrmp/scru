description: Fix PR review comments and re-wrap the task
---

Fix the comments on the open draft PR, then wrap the task again using the updated work.

Use the `review-fix-task` skill.

Requirements:

- Resolve the requested PR review comments.
- Make the smallest correct changes.
- Inspect the changes made by the fix.
- Summarize the fixes.
- Update the task spec if needed.
- Save any important technical detail to memory.
- Commit the fixes to the current branch.
- Refresh the draft pull request.

If any review comment is ambiguous, ask for clarification before changing code.
